Router On A Stick (ROAS)

Definition

Router-on-a-Stick (ROAS) is a networking technique used to enable communication between multiple VLANs using a single physical router interface.

It is called "stick" because only one physical cable connects the router to the switch while carrying traffic from multiple VLANs.

Lab Scenario

In this lab, I created a multi-switch VLAN environment with Inter-VLAN Routing using Router-on-a-Stick.

  • 🟦 VLAN 10 → IT Department
  • 🟨 VLAN 20 → Accounts Department

Two switches were connected using a trunk link, and a Cisco router was configured with subinterfaces to route traffic between VLANs.

IP Addressing Scheme

Department VLAN Network Gateway
IT 10 192.168.10.0/24 192.168.10.1
Accounts 20 192.168.20.0/24 192.168.20.1

How It Works

Switch Side

The switch ports connected between switches and router were configured as trunk ports. 

interface fa0/5
 switchport mode trunk

interface fa0/6
 switchport mode trunk

This allows VLAN traffic from multiple VLANs to travel through a single connection.

Router Side

The router uses subinterfaces to act as default gateways for each VLAN. 

interface fa0/0.10
 encapsulation dot1Q 10
 ip address 192.168.10.1 255.255.255.0

interface fa0/0.20
 encapsulation dot1Q 20
 ip address 192.168.20.1 255.255.255.0
  • Each subinterface is linked to a VLAN.
  • The router performs routing between VLANs.
  • Devices use router IP as default gateway.

What I Configured

VLAN Creation

vlan 10
 name IT

vlan 20
 name Accounts

Access Port Configuration

interface range fa0/1 - 2
 switchport mode access
 switchport access vlan 10

interface range fa0/3 - 4
 switchport mode access
 switchport access vlan 20

Trunk Configuration

interface fa0/5
 switchport mode trunk

interface fa0/6
 switchport mode trunk

Testing Results

  • ✅ Devices within same VLAN communicated successfully
  • ✅ VLAN 10 devices communicated with VLAN 20 devices through router
  • ✅ Inter-VLAN routing worked correctly

Key Learning

  • ROAS uses one physical router interface for multiple VLANs
  • 802.1Q tagging is required for VLAN identification
  • Subinterfaces act as gateways for VLANs
  • Trunk ports carry traffic for multiple VLANs
  • Inter-VLAN communication requires Layer 3 routing

Why Router-on-a-Stick Is Important

  • Reduces cabling complexity
  • Uses fewer router interfaces
  • More scalable than legacy inter-VLAN routing
  • Widely used in small and medium enterprise networks
