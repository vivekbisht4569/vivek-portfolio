📘 What is Router On A Stick?

Router-on-a-Stick (ROAS) is a networking technique used to allow communication between multiple VLANs using a single physical router interface.

Instead of using separate router interfaces for each VLAN, a single interface is divided into multiple logical subinterfaces using 802.1Q encapsulation.

🧪 Lab Scenario

In this lab, I created a multi-switch VLAN environment where devices from different departments communicate through a router using Inter-VLAN Routing.

🟦 VLAN 10

IT Department Network

🟨 VLAN 20

Accounts Department Network

🌐 Goal

Enable communication between different VLANs using Router-on-a-Stick.

🌍 IP Addressing Scheme

Department VLAN Network Gateway
IT 10 192.168.10.0/24 192.168.10.1
Accounts 20 192.168.20.0/24 192.168.20.1

🔧 Switch Configuration

VLAN Creation

vlan 10
 name IT

vlan 20
 name Accounts

Access Port Configuration

interface range fa0/1 - 2
 switchport mode access
 switchport access vlan 10

interface range fa0/3 - 4
 switchport mode access
 switchport access vlan 20

Trunk Port Configuration

interface fa0/5
 switchport mode trunk

interface fa0/6
 switchport mode trunk

🛜 Router Configuration

The router uses subinterfaces to route traffic between VLANs. 

interface fa0/0.10
 encapsulation dot1Q 10
 ip address 192.168.10.1 255.255.255.0

interface fa0/0.20
 encapsulation dot1Q 20
 ip address 192.168.20.1 255.255.255.0

✅ Lab Outcome

✅ VLAN Communication

Devices inside the same VLAN communicated successfully.

✅ Inter-VLAN Routing

Devices from VLAN 10 and VLAN 20 communicated through the router.

✅ Trunk Verification

802.1Q trunk links carried multiple VLAN traffic correctly.
