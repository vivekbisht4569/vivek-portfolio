🦈 Packet Analysis & Reconnaissance

Hands-on packet analysis focused on reconnaissance detection, TCP communication analysis, and packet inspection using Kali Linux, Wireshark, and Nmap inside an isolated virtual environment.

📡 Nmap Reconnaissance Scan

Performed reconnaissance scanning from Kali Linux against the Windows host system using Nmap to generate TCP traffic and observe scanning behavior inside Wireshark.

nmap 192.168.56.1
Nmap Reconnaissance TCP Scan Kali Linux
Outcome:

Generated multiple TCP SYN packets targeting different ports on the Windows host system. This demonstrated how reconnaissance activity appears during port scanning operations.

🦈 ICMP Packet Capture

Captured ICMP traffic generated during ping communication between Kali Linux and the Windows host system using Wireshark packet analysis.

icmp
Wireshark ICMP Packet Capture Ping Analysis
Outcome:

Observed Echo Request and Echo Reply packets exchanged between both systems, confirming successful communication and packet visibility.

🌐 TCP Traffic Analysis

Analyzed TCP communication generated during reconnaissance scanning and observed TCP connection attempts using Wireshark filters.

tcp
TCP Wireshark Network Analysis Traffic Monitoring
Outcome:

Observed TCP traffic including SYN packets, connection attempts, and source-to-destination communication during scanning activity.

🚨 SYN Scan Detection

Used Wireshark filters to isolate TCP SYN packets generated during Nmap reconnaissance activity.

tcp.flags.syn == 1
SYN Scan Recon Detection Packet Analysis Wireshark
Outcome:

Successfully identified rapid SYN packet generation across multiple destination ports, demonstrating detectable reconnaissance behavior.
