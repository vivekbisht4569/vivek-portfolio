🛡️ SOC Home Lab

Hands-on cybersecurity home lab focused on SOC analysis, packet investigation, attack simulation, Windows log monitoring, and detection engineering using Kali Linux, Windows 11, Wireshark, and VirtualBox in an isolated sandbox environment.

🪟 LAB 1 — Windows Event Log Analysis & Nmap Detection

Built an isolated sandbox lab using Kali Linux and Windows 11 to simulate reconnaissance activity using Nmap and analyze Windows Event Viewer logs for inbound traffic detection and event correlation.
