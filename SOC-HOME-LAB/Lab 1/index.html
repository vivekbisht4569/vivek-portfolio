SOC Home Lab

Overview

A practical SOC & Detection Engineering home lab built using Kali Linux, Windows 11, and VirtualBox in an isolated sandbox environment. This lab focuses on attack simulation, Windows event log analysis, network monitoring, and blue team investigation workflows.

Lab Setup

  • Host OS: Windows 11
  • Attacker Machine: Kali Linux VM
  • Virtualization: VirtualBox
  • Network Type: Host-Only Adapter
  • Environment: Isolated Sandbox

Attack Simulation

Performed reconnaissance and TCP scan using Nmap from Kali Linux to the Windows host machine. 

nmap -Pn -sT -p 80 192.168.56.1

Detection & Analysis

  • Analyzed Windows Event Viewer logs
  • Investigated Event ID 5156 & 5157
  • Detected inbound attacker traffic
  • Identified source IP and destination ports
  • Compared normal vs suspicious traffic behavior
